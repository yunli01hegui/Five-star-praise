# mouse over（鼠标悬浮）
When the mouse hovers over the five-pointed star at the specified position, the five-pointed star and the previous five-pointed star will be completely filled(当鼠标悬浮在指定位置的五角星上，该五角星以及前面的五角星都将被填充完全)
example（举例）：The mouse hovers over the third five-pointed star（鼠标悬浮在第三颗五角星上）
